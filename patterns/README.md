# Go Patterns

Структура для изучения паттернов проектирования Go.

## Категории

- **creational** - порождающие паттерны (Singleton, Factory, Builder, etc.)
- **structural** - структурные паттерны (Adapter, Bridge, Composite, etc.)
- **behavioral** - поведенческие паттерны (Strategy, Observer, Command, etc.)
- **concurrency** - паттерны конкурентности (Worker Pool, Pipeline, Fan-out, etc.)

Создавайте подпапки для каждого паттерна с README и кодом.
